# ❌⭕ Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**. Two players can take turns marking X and O on a 3×3 grid, with automatic win and draw detection.

---

## 🎮 Features

- ❌ Two-player gameplay (X vs O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart/New Game functionality
- 📱 Responsive design
- 🎨 Clean and modern user interface
- ⚡ Built using vanilla HTML, CSS, and JavaScript

```bash

```

3. Open `index.html` in your preferred web browser.

---

## 🎯 Game Rules

- Player **X** starts the game.
- Players take turns placing their marks on the board.
- The first player to align **three marks** horizontally, vertically, or diagonally wins.
- If all cells are filled without a winner, the game ends in a draw.
